## Upgrade Instructions

Follow these steps to upgrade your Test platform instance safely.

<Steps>
  <Step title="Backup Data" icon="database">

  Create a full backup of your database and files.

  ```bash
  pg_dump -U user test_db > test_backup_$(date +%Y%m%d).sql
  tar -czf files_backup_$(date +%Y%m%d).tar.gz /path/to/files
  ```

  </Step>
  <Step title="Update Package" icon="package">

  <Tabs>
    <Tab title="npm" icon="nodejs">
    ````bash
    npm install test@latest
    ````
    </Tab>
    <Tab title="Docker" icon="docker">
    ````bash
    docker pull test/platform:latest
    docker-compose up -d
    ````
    </Tab>
  </Tabs>

  </Step>
  <Step title="Run Migrations" icon="git-branch">
    Apply database migrations.
    ```bash
    npm run migrate
    ```
  </Step>
  <Step title="Verify" icon="check-circle">
    Test key endpoints.
    ```bash
    curl -H "Authorization: Bearer YOUR_TOKEN" https://api.example.com/health
    ```
  </Step>
</Steps>
