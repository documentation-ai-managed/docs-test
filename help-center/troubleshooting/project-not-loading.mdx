---
title: Project Not Loading
description: Quick fix for project load issues.
---

## Symptoms

- Project page shows error or blank
- Can't access project dashboard

## Causes

- Browser cache
- Network problem

## Solution

<Steps>
  <Step title="Clear browser cache">
    ````
    On Chrome: Settings > Privacy > Clear browsing data
    ````
  </Step>
  <Step title="Check network">
    Ensure your internet connection is stable.
  </Step>
</Steps>
